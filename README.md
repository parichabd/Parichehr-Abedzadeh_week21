## About The Project

![Product Name Screen Shot](./Screenshots/hal-gatewood-weRQAu9TA-A-unsplash.jpg)

This project is a product management dashboard built with Next.js. It allows users to register and sign in, manage products through full CRUD operations, search products, and navigate data using pagination. The application includes protected routes to restrict access to authenticated users only and provides a clean, user-friendly interface for managing products efficiently.

  ```sh

  ```

2. Install NPM packages
   ```bash
   npm install
   ```

